# Contracts: Knowledge Pipeline

Este diretório agrupa contratos de API e mensagens usados no Knowledge Pipeline.

## Arquivos sugeridos

- `rest-api.yaml` – OpenAPI/Swagger para endpoints `/api/knowledge/*`.
- `agents-contracts.md` – Esquemas de entrada/saída dos agentes Agno.
- `events.md` – Eventos assíncronos (se adotarmos fila ou pub/sub).

Por enquanto, este arquivo serve apenas como marcador; contratos detalhados serão adicionados quando o desenho da API for congelado.
